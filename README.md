Além dos exercícios propostos, também desenvolvi projetos totalmente autorais (amanda-secrets, franciele-santiago e locomotivos), sempre respeitando os conceitos trabalhados ao longo das aulas.

Este repositório representa minha evolução nos fundamentos do desenvolvimento Front-end e registra minha trajetória de aprendizado durante essa etapa da formação.
